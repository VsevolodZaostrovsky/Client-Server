Проверка возможности создания устройства: 
ISPOSSIBLE NAME="что" QUANTITY="сколько" END
Команда имеет вид ISPOSIIBLE, после чего указывается пара: 
имя устройства - количество штук. Слово END заканчивает запрос.

Сколько устройств данного типа можно собрать:
HOWMANY NAME="что" END

Удаление детали (команда не сработает, если есть деали, для сборки которых необходима эта):
DELITEM NAME="что" END

Уменьшение количества деталей: 
DEL NAME="имя" QUANTITY="количество" END

Добавление новой детали на склад (если деталь с таким именем уже существует, то она будет заменена):
ADDITEM NAME="имя" QUANTITY="количество" CREATED="возможно ли создать" COMP="имя" QUANTITY="количество компонентов с таким именем" ... END 

Увеличение количества деталей:
ADD NAME="имя" QUANTITY="количество" END

Сборка детали:
CREATE NAME="имя" QUANTITY="количество" END

Печать карты сборки:
PRINT NAME="имя" END

Редактирование сборочной карты:

ADDCOMP NAME="имя" COMP="имя" QUANTITY="количество компонентов с таким именем" ... END

DELCOMP NAME="имя" COMP="имя" ... END
